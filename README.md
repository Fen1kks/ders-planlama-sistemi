# ders-planlama-sistemi
Mühendislikteki karmaşık ön koşul zincirlerini statik PDF'ler yerine interaktif bir ağaç yapısında görselleştiren bu proje; öğrencilerin ders seçimlerinde zincirleme hatalar yapmasını engeller. Sistem, öğrencilere sadece anlık GPA hesabı değil, mezuniyete giden yolda stratejik ve güvenilir bir rota çizme imkanı tanır.
